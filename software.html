Software de Desarrollo SmartPill

Explora las herramientas digitales utilizadas para crear la aplicación, gestionar datos y controlar el dispositivo SmartPill.

Android Studio

Escogimos Android Studio porque facilita el desarrollo de aplicaciones móviles, permitiendo integrar bases de datos, notificaciones y conexión con dispositivos externos en una sola plataforma de forma eficiente y segura.

Firebase

Utilizamos Firebase para almacenar y administrar la información de los usuarios, medicamentos e historial, permitiendo guardar los datos de forma segura, sincronizada y accesible en tiempo real desde la aplicación.

Arduino IDE

Utilizamos Arduino IDE para programar el ESP32, ya que facilita la configuración y control del hardware, permitiendo gestionar sensores, LEDs, alarmas y la comunicación con la aplicación de manera sencilla y eficiente.

Visual Studio Code

Utilizamos Visual Studio Code para desarrollar la página web, ya que permite programar de forma rápida y organizada utilizando HTML, CSS y JavaScript, facilitando el diseño, edición y visualización del sitio web de SmartPill.
