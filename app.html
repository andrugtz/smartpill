Aplicación Inteligente SmartPill

Descubre las herramientas y funcionalidades diseñadas para mejorar la organización, supervisión y seguimiento del tratamiento médico.

Inicio de Sesión y Registro

Accede de forma segura a SmartPill con tu correo y contraseña. Si no tienes una cuenta, puedes crearla fácilmente para gestionar medicamentos y recordatorios personalizados.

Selección de Perfil

Elige el tipo de perfil que mejor se adapte a tus necesidades: Paciente o Cuidador. Cada opción brinda funciones personalizadas para facilitar el seguimiento, control y administración de medicamentos de manera eficiente.

Menú Principal

Accede rápidamente a las principales funciones de SmartPill desde su menú. Consulta el inicio, administra medicamentos, revisa el historial, personaliza la configuración y accede a instrucciones para un mejor uso.

Inicio

Visualiza de manera rápida la información más importante del día, incluyendo la cantidad de medicamentos programados, el último medicamento registrado, el estado del pastillero y el progreso diario de las tomas realizadas.

Medicamentos

Administra fácilmente cada tratamiento registrando el nombre del medicamento, dosis, frecuencia, horario de inicio, duración y notas adicionales. Además, permite identificar cada medicamento mediante colores y ofrece opciones para registrar o eliminar información.

Historial y Gráficas

Consulta el seguimiento de medicamentos mediante una gráfica que muestra las tomas realizadas y las omitidas. Además, visualiza un historial detallado con la lista de medicamentos programados y el estado de cada uno, indicando si fueron tomados o no.

Configuración

Personaliza la aplicación modificando la información de tu perfil, como nombre de usuario, correo electrónico y el modo de uso, permitiendo cambiar entre paciente o cuidador según tus necesidades.
